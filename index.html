Guide de Personnalisation

Comment éditer et personnaliser votre portfolio GitHub Pages

Édition de Base

1 Accéder au fichier

Pour modifier votre site :

  1. Allez sur votre repository GitHub
  2. Cliquez sur le fichier index.html
  3. Cliquez sur l'icône (éditer)

2 Modifier le contenu

Recherchez ces sections dans le code :

Section À propos

<h2>Développeur Web Fullstack</h2> <p>Je suis un développeur web avec 3 ans d'expérience...</p>

Coordonnées

<span>contact@yiadh.fr</span> <span>+33 6 12 34 56 78</span>

3 Sauvegarder les modifications

Après avoir fait vos modifications :

  1. Descendez en bas de la page
  2. Ajoutez un titre descriptif pour votre modification
  3. Cliquez sur le bouton "Commit changes"
  4. Attendez 1-2 minutes pour que le site se mette à jour

Personnalisation Avancée

Changer les couleurs

Modifiez les variables CSS en haut du fichier :

#6f42c1 Primaire
#2c3e50 Secondaire
#ff6b6b Accent
#2ecc71 Succès
:root { --primary: #6f42c1; /* Couleur principale */ --secondary: #2c3e50; /* Couleur secondaire */ --accent: #ff6b6b; /* Couleur d'accent */ --success: #2ecc71; /* Couleur de succès */ }

Modifier la photo de profil

Remplacez l'URL dans la section À propos :

<img src="https://images.unsplash.com/photo-1507003211169-0a1dd7228f2d?ixlib=rb-4.0.3..." alt="Photo de Yiadh">

Conseil : Utilisez Unsplash pour trouver des images libres de droits, ou uploader la vôtre sur un service comme Imgur.
